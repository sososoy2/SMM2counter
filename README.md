# SMM2 COUNTER!
複数カウンターのマリメ特化版  
勝率を自動で計算してくれます。  

### 使い方
「試合数」「勝ち」「本負け」「ゴール負け」「無効試合」「ギブアップ」の6種類のカウンターが表示されます。  
カウンターはボタンで増減させることができます。  
みんなでバトルの結果に合わせて、ボタンを押してください。
