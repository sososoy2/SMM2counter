# SMM2 COUNTER!
複数カウンターのマリメ特化版  
勝率を自動で計算してくれます。  

### 使い方
「試合数」「勝ち」「本負け」「ゴール負け」「無効試合」「ギブアップ」の6種類のカウンターが表示されます。  
みんなでバトルの結果に合わせて、カウンターボタンを押してください。  
編集機能もあるので、自分好みにカスタマイズできます。
良いみんバトライフを！

### OBS用のカウンターについて
基本的に使い方は通常版と同じですが、配信向けに一部機能が最適化されています。
### 導入方法  
1. OBSを開く
2.「ソース」の [+] ボタンを押す  
3. [ブラウザ] を選択  
4. URLに https://sososoy2.github.io/SMM2counter/obs.html と入力
5. OKを押す
